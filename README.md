# CLUSTERSTRIKE CREW GAME INSTALL!!!

This repo is for the Clusterstrike Crew (my buds) to synchronize our mekhq builds so we can use git to synchronize our saves and custom info.

You'll want to use some kind of Git tool to stay synchronized and resolve merge conflicts.

Github has a desktop app that does this: https://desktop.github.com/download/

If you want to do something with the campaign file, do things in this order:

1. PULL FROM REMOTE
2. LOAD THE SAVE FILE
3. MAKE YOUR CHANGES WITHIN MEKHQ
4. SAVE OVER THE FILE
5. COMMIT YOUR CHANGES - ADD A MESSAGE EXPLAINING WHAT YOU DID
6. PUSH BACK UP TO REMOTE
